About AvstroGroup

AvstroGroup is created to make the creation, maintenance, and payment of car insurances easy.

With a couple of clicks, you can create and modify your insurances! No need to go to an office, no need to wait there for half an hour.

This project is created as an assignment from Softuni for their Angular course! This website is NOT official, and it is just a demonstration of my current skills in angular!
